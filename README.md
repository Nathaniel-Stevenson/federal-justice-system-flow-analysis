# federal-justice-system-flow-analysis

Dataset compiled from data provided by Department of Justice. Source: https://www.bjs.gov/fjsrc/

The Charge–Arrest Net Flow is driven primarily by changes in persons arrested and booked, not by changes in persons charged. Persons charged stays relatively stable for most of the period, generally sitting in the 70k–100k range, while arrests swing much more sharply, from about 78k to over 206k. Because net flow is calculated as charged minus arrested, years with higher arrests produce much larger negative net flows.
